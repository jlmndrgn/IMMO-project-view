# IMMO-project-view
Projektien aikataulutustyökalu

Gantt-työkalu usean projektin hahmottamiseen vuosikontekstissa. 
- Kalenteri näyttää 52-viikkoisen vuoden, jonka aloituskuukauden voit vaihtaa.
- Muokkausmoodissa voit:
- - tallentaa ja ladata eri projektikonfiguraatioita,
- - muokata projektien määrää, nimiä, alkuja, vaiheiden nimiä, kestoja ja limittymistä,
- - korostaa esim. organisaation lomaviikot punaisella taustavärillä (Off-weeks).
- Klikkaamalla työvaiheen palkkia voit merkitä viikon tarkkuudella geneerisiä tapahtumia, esim. virstanpylväitä.
